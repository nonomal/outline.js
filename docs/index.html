outline.js

outline.js - 自动生成文章导读（Table of Contents）导航的 JavaScript 工具。

<script src="https://cdn.jsdelivr.net/gh/yaohaixiao/outline.js/outline.min.js"></script>
Download Zip

Options

anchorAt

Description

Type：
String
Default：
'front'

可选，用来指定文章标题锚点链接图标的显示位置。

// // front - 默认锚点图标显示在标题最开始位置
new Outline()

// end - 锚点图标显示在标题最后面
new Outline({
    anchorAt: 'end'
})

anchorURL

Description

Type：
String
Default：
''

可选，用来指定文章标题锚点链接图标的链接地址。

// Default URL - 默认链接，点击图标，页面会滚动到当前标题位置
new Outline()

// Customize URL - 直接打开链接了
new Outline({
    anchorURL: 'http://www.yaohaixiao.com/'
})

article

Description

Type：
String
Default：
'#article'

必选，用来指定页面中显示文章正文的 DOM 节点的 ID 选择器。

// Default Selector
new Outline()

// Customize Selector
new Outline({
    article: $('#main')
})

isGenerateHeadingAnchor

Description

Type：
Boolean
Default：
true

可选，用来指定是否在文章标题位置生成锚点链接图标。

isGenerateHeadingChapterCode

Description

Type：
Boolean
Default：
false

可选，用来指定是否在文章标题前面显示文章段落层次的索引编号。

isGenerateOutline

Description

Type：
Boolean
Default：
true

可选，用来指定是否文章导读导航菜单。

isGenerateOutlineChapterCode

Description

Type：
Boolean
Default：
true

可选，用来指定是否文章导读导航菜单是否显示文章段落层次的索引编号。

selector

Description

Type：
String
Default：
'h1,h2,h3,h4,h5,h6'

可选，用来指定 article 节点下，要生成导航的标题标签的选择器。

// Default selector
new Outline()

// Customize selector
new Outline({
    selector: 'h3'
})

title

Description

Type：
String
Default：
'文章导读'

可选，用来指定文章导读导航菜单的标题文字。

// 不配置 title 属性，即使用默认标题文字
new Outline({
    article: '#article'
})

// 自定义标题
new Outline({
    title: 'outline.js API'
})

position

Description

Type：
String
Default：
'outside'

可选，用来指定文章导读导航菜单的显示位置。

// Default position - （'inside'）默认以侧边栏导航的形式显示
new Outline()

// Inside Article
new Outline({
    position: 'inside'
})

Properties

defaults

Description

Type：
Object

静态属性，存储的是 Outline 对象默认配置选项：

  • article：
    用来指定页面中显示文章正文的 DOM 节点的 ID 选择器。（默认值：'#article'）
  • selector：
    用来指定 article 节点下，要生成导航的标题标签的选择器。（默认值：'h1,h2,h3,h4,h5,h6'）
  • title：
    用来指定文章导读导航菜单的标题文字。（默认值：'文章导读'）
  • position：
    用来指定文章导读导航菜单的显示位置。（默认值：'outside'）
  • anchorURL：
    用来指定文章标题锚点链接图标的链接地址。（默认值：''）
  • anchorAt：
    用来指定文章标题锚点链接图标的显示位置。（默认值：'front'）
  • isGenerateOutline：
    用来指定是否文章导读导航菜单。（默认值：true）
  • isGenerateOutlineChapterCode：
    用来指定是否文章导读导航菜单是否显示文章段落层次的索引编号。（默认值：true）
  • isGenerateHeadingChapterCode：
    用来指定是否在文章标题前面显示文章段落层次的索引编号。（默认值：false）
  • isGenerateHeadingAnchor：
    用来指定是否在文章标题位置生成锚点链接图标。（默认值：true）

attributes

Description

Type：
Object

存储的是 Outline 对象当前使用中的配置选项：

说明：建议使用 get(prop) 方法来获取属性，避免直接调用属性。

elements

Description

Type：
Object

存储的是 Outline 对象相关的 DOM 元素：

说明：建议使用 getElements() 方法来获取 elements 属性。

data

Description

Type：
Object

存储的是根据标题 DOM 元素分析的数据：

chapters：
标题章节索引数据，单个的 chapter 数据格式如下：
{
   // 段落章节编号
   code: "1.2.3",
   // 标题ID
   id: 0,
   // 标题在父章节中的索引值
   index: 1,
   // 标题的级别
   level: 1,
   // 父标题的ID
   pid: -1,
   // 标题链接地址
   rel: "outline-heading-1",
   // 标题文字
   text: "Outline.js"
}
headings：
各个标题相关的数据：
{
    // 标题节点对应的ID
    id: "outline-heading-1",
    // 标题级别
    level: 1,
    // 标题节点 DOM 对象
    node: h1#outline-heading-1.title.outline-heading,
    // 标题的标签名
    tagName: "H1"
}

说明：建议使用 getData() 方法来获取对应的数据。

Methods

get ( prop )

Description

get ( prop ) 方法返回某个 attributes 属性。

Parameters

prop
Type：
String

（必选）想要获取的 attributes 属性名称。

Returns

Type：
String|Boolean

attributes 属性值。

const outline = new Outline({
     article: '#article',
     title: 'Outline v2.0.0'
})

outline.get('title') // -> 'Outline v2.0.0'

scrollTo ( top )

Description

scrollTo ( top ) 方法用于将窗口的滚动条滚动到指定 top 值的位置。

Parameters

top
Type：
Number

（必选）页面要滚动的 top 数值。

Returns

Type：
Outline

Outline 对象。

const outline = new Outline({
     article: '#article',
     title: 'Outline v2.0.0'
})

// -> 页面将滚动到距离顶部 200 像素的位置
outline.scrollTo(200)

stop()

Description

停止页面滚动：

  1. 绘制文章开始出的目录导航；
  2. 绘制标题栏的锚点链接和标题段落索引编码；
  3. 绘制侧边栏的目录导航菜单；

Returns

Type：
Outline

Outline 对象。

const outline = new Outline({
   article: '#article',
   title: 'Outline v2.0.0'
})

// -> 页面将滚动到距离顶部 200 像素的位置
outline.scrollTo(200)

// -> 手动停止滚动，点击不同的标题链接时就会先停止滚动，然后再滚动到目标位置
outline.stop()

toggle()

Description

toggle() 方法用来展开侧或者收起边栏菜单。

remove()

Description

remove() 方法将删除所有 Outline 创建的 DOM 节点，同时取消所有的事件绑定。

Returns

Type：
Outline

Outline 对象。

const outline = new Outline({
     article: '#article',
     title: 'Outline v2.0.0'
});

// -> 将删除所有 Outline 创建的 DOM 节点
outline.remove();

destroy()

Description

destroy() 方法用于移除所有绘制的 DOM 节点，并移除绑定的事件处理器，同时重置所有 attributes 配置信息和 data 信息。

Returns

Type：
Outline

Outline 对象。

const outline = new Outline({
     article: '#article',
     title: 'Outline v2.0.0'
});

outline.destroy();

reload ( options )

Description

reload ( options ) 是（根据新的配置信息）重启程序方法：

  1. 先移除所有绘制的 DOM 元素和绑定的事件处理器；
  2. 重新初始化程序；

Parameters

options
Type：
Object

（可选）初始化的配置参数。

Returns

Type：
Outline

Outline 对象。

const outline = new Outline({
    isGenerateHeadingAnchor: true
})

// 鼠标滑过正文中的标题上将不会创建锚点链接图标
outline.reload({
    isGenerateHeadingAnchor: false
})
